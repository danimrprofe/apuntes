# Zumbador

Un zumbador (también conocido como un buzzer) es un dispositivo electromecánico que **produce un sonido** audible. Se utiliza en muchos sistemas electrónicos para indicar una señal, como una **alarma**, una advertencia o una confirmación de acción.

## Tipos de zumbadores

Se pueden categorizar en **activos** y **pasivos**

- Un [zumbador activo](zumbador-activo.md) tiene una fuente oscilante integrada, por lo que va a generar un sonido cuando se electrifica. El  zumbador activo es a menudo más caro.
- Un [zumbador pasivo](zumbador-pasivo.md)  no tiene esa fuente para que no pitan si se utilizan señales de DC; en cambio,  necesitará usar ondas cuadradas cuya frecuencia es entre 2K y 5K manejarlo.
